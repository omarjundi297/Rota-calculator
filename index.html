About This Calculator

This tool uses a validated model to predict the Whole Time Equivalent (WTE) staffing required to run a stable rota. The final prediction is based on the Minimum WTE plus the calculated WTE Cost of Inefficiency.

  • WTE Cost of Inefficiency: This is the extra WTE required to overcome workforce fragmentation. It is calculated using a quadratic model that accurately predicts the small number of gaps in nearly full-time systems while also capturing the large impact in highly fragmented teams.
  • System Multiplier: The type of rostering system determines the true cost of fragmentation. A pooled system is most efficient at handling it (lowest cost), while a rigid, siloed system is the least efficient (highest cost). This is factored into the calculation.
